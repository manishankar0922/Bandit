# Bandit

Bandit is a web application that allows users to play the "One Armed Bandit" (slot machine) game.

## Features

- **Play Game**: Spin the reels and try to match symbols to win.
- **Simple UI**: Clean and intuitive interface for playing the game.
- **Responsive Design**: Works on different screen sizes.

## Setup

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <repository-url>
   cd Bandit
   ```

2. **Install dependencies** (if needed):
   ```bash
   npm install
   ```

## Usage

To start the application:

```bash
npm start
```

The application will open at `http://localhost:5173`.
